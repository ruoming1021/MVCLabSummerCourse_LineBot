# MVCLabSummerCourse_LineBot

* **How to run**
    * **Step 1: Install Python Packages**
        * > pip install -r requirements.txt
    * **Step 2: Run ngrok**
        * > ngrok http 1234
    * **Step 3: Test LineBot**
        * > python count.py
